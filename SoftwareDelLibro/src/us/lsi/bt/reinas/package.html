El Problema de las N Reinas es un problema clásico que puede ser resuelto por diversas técnicas algorítmicas.

El problema parte de un tablero de ajedrez vacío de dimensión N y se trata de colocar N Reinas sin que se viole ninguna regla del ajedrez

Una Reina puede ser modelada como un objeto con las propiedades:

  • x, Coordenada x donde está ubicada la reina con valor de 0 a N-1
  • y, Coordenada y donde está ubicada la reina con valor de 0 a N-1
  • dp, Diagonal principal donde está ubicada la reina. dp = y-x
  • ds, Diagonal secundaria donde está ubicada la reina. dp = y+x

La clase {@link us.lsi.bt.reinas.TableroDeReinas TableroDeReinas} representa tablero de ajedrez de tamaño n variable con n reinas colocadas. Estas reinas pueden o no violar las reglas. Esta clase tiene las propiedades:

  • NumDeReinas, El número de reinas en el tablero igual a su dimensión
  • YOcupadas, Una lista con las filas ocupadas
  • DiagonalesPrincipalesOcupadas, Un conjunto con las diagonales principales ocupadas
  • DiagonalesSecundariasOcupadas, Un conjunto con las diagonales secundarias ocupadas
  • Objetivo, Un número que indica si hay violaciones de las reglas. No hay si objetivo es cero.

Las clases disponibles en el paquete son:

  • La clase {@link us.lsi.bt.reinas.Reina Reinas} representa los objetos que se van a incluir en la mochila.
  • La clase {@link us.lsi.bt.reinas.TableroDeReinas TableroDeReinas} representa tablero de ajedrez de tamaño n variable con n reinas colocadas
  • La clase {@link us.lsi.bt.reinas.EstadoReinasBT EstadoReinasBT} implementa estado del problema para ser resuelto mediante BactTracking
  • La clase {@link us.lsi.bt.reinas.EstadoReinasSA EstadoReinasSA} implementa estado del problema para ser resuelto mediante Simulated Annealing
